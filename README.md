# 데이터 분석 프로젝트
### 1. 패키지 설치
<code>pip install -r requirements.txt</code>   
해당 프로젝트 터미널에서 위 코드 실행
### 2. .env 파일 생성
터미널에서 <code>copy .env.example .env</code>
> 예시  
> Flask  
> FLASK_APP=app.py  
> FLASK_ENV=development  
> SECRET_KEY=randomstringofcharacters  
> Mysql  
> MYSQL_USER=root  
> MYSQL_PASSWORD=123456  
> MYSQL_DB=kosa-3
